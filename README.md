# Blog Management System

## Overview

A secure Blog Management System developed using PHP and MySQL as part of the ApexPlanet Web Development Internship.

## Features

* User Registration
* User Login & Logout
* Password Hashing
* Session Management
* Create, Read, Update, Delete (CRUD) Operations
* Search Functionality
* Pagination
* Bootstrap Responsive UI
* Admin and Editor Roles
* Role-Based Access Control
* Server-Side Validation
* Client-Side Validation
* Prepared Statements for SQL Security

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* Git
* GitHub
